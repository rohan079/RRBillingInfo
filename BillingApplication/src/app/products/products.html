Generate barcodes

Enter how many products you want (e.g. 50). Each gets a unique barcode like AUTO-1730000000000-00001 using your prefix and a batch id.

Remove barcodes by count

Enter how many rows to remove (e.g. 40 after creating 50). Uses the same prefix as generate (default AUTO). Without a batch id, the newest matching rows are removed first. Optional batch id is the middle number in AUTO-1730000000000-00001 so only that batch is touched; then highest suffix numbers are removed first.
